ポイント管理Webアプリ

HTML CSS JavaScript LocalStorage

概要

子どものお小遣い管理をきっかけに制作したポイント管理Webアプリです。 現在はタスク管理や目標管理など、利用者ごとに柔軟な使い方ができる 汎用的なアプリとして継続開発しています。

制作背景

ポイントカードの紛失やポイント管理の煩雑さを解決するため、 実生活の課題から着想して制作しました。

主な機能

  • ユーザー・項目の追加／編集／削除
  • ポイント加算・減算・支給
  • 個人目標・共有目標
  • 履歴表示・データ永続化

工夫した点

  • 利用者ごとに使い方を変えられる柔軟な設計
  • 進捗バーによる直感的な可視化
  • 目標機能によるモチベーション維持

Salesforce試験対策アプリ（GAS）

Google Apps Script JavaScript HTML/CSS Google Sheets

概要

Salesforce資格試験の学習効率向上を目的として制作したWebアプリです。 購入した問題集をより実践的に活用できるよう、ランダム出題や誤答問題の復習機能を実装し、外出先でも学習できる環境を構築しました。

制作背景

問題集が長文形式だったため、スクロール中に答えが見えてしまうことや、外出先で学習しづらいことが課題でした。 また、正答率の把握やランダム出題による反復学習を行える環境を作りたいと考え、本アプリを制作しました。

主な機能

  • ランダム出題
  • 誤答問題のみの再出題
  • 正誤判定・解説表示
  • 正答率表示
  • 回答時間表示
  • Google Sheetsから問題データ取得

工夫した点

  • Google Sheets上で問題を管理できる構成
  • コードを変更せず問題を追加・更新可能
  • スマートフォンでも学習しやすいシンプルなUI
  • 隙間時間でも効率的に学習できる設計

Game Portfolio

HTML CSS JavaScript

概要

HTML・CSS・JavaScriptを用いて制作したゲーム風ポートフォリオです。 作品を掲載するだけでなく、自分らしい発想やUI/UXへの考え方も伝えられるポートフォリオを目指し、キャラクターを操作して各コンテンツへ移動するゲームライクなUIを採用しました。

制作背景

一般的なポートフォリオとは異なるアプローチで、自分らしさや発想力を表現したいと考え、「操作する楽しさ」も体験できるサイトを目指して制作しました。

主な機能

  • RPG風ホーム画面
  • キャラクター移動（WASD・矢印キー・スマホ操作対応）
  • 建物クリック・入口Hit Areaによるページ遷移
  • スプライトアニメーション
  • フェードアニメーション
  • モーダルによる作品紹介
  • レスポンシブ対応

工夫した点

  • 建物へのクリック・接触どちらでも操作できるUI設計
  • 各ページの世界観を統一したデザイン
  • ゲームらしい操作体験を意識したインタラクション
